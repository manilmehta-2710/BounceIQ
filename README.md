# BounceIQ
An AI-powered web application that predicts user bounce rate and exit probability based on session behavior and interaction data. Built to help businesses optimize user engagement, improve retention, and identify high-risk drop-off points.

🚀 Features

* Predicts bounce likelihood for active sessions
* Identifies pages with high exit probability
* Machine learning model (XGBoost / Gradient Boosting fallback)
* Real-time analytics via Flask API
* SHAP-based feature importance for explainability
* Batch prediction support for large datasets

🛠️ Tech Stack

* Backend: Flask (Python)
* ML Models: XGBoost / Scikit-learn
* Frontend: HTML, CSS, JavaScript
* Visualization: Matplotlib / SHAP

📈 Use Cases

* Website performance optimization
* UX/UI improvement insights
* Marketing funnel analysis
* Conversion rate enhancement

⚙️ How It Works

User session data → Feature processing → ML prediction → Explainability (SHAP) → Insights dashboard
